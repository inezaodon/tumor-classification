### Branch and Pull Instructions
**Before starting work, pull latest**
git pull origin main

**Create a branch for your feature**
git checkout -b feature/your-name-task

**After changes, stage and commit**
git add .
git commit -m "Add preprocessing pipeline"

**Push and open a PR on GitHub**
git push origin feature/your-name-task

### Install Requirements
**After cloning the repository, open your terminal and run the following command to install all required dependencies:**
After cloning the repository, open your terminal and run the following command to install all required dependencies:

Note: Check python version `python3 --version`. Needs to be 3.11 or 3.12

1. Create new python environment to install reqs: Navigate to project folder in terminal and run `python3 -m venv .venv` or `python3.12 -m venv .venv` (make sure your python version is 3.11 or 3.12 to use some packages)
2. Activate virtual env: `source .venv/bin/activate`
3. Install requirements: `pip install -r requirements.txt`
4. When you want to exit the env to work on something else, type `deactivate`
